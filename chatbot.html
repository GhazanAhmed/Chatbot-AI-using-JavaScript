Chatbot

In this project, you will create a chatbot that interacts with the user in the browser using the OpenAI's v1/chat/completions endpoint and GPT-3.5-turbo model.

Requirements:

  1. Use the JavaScript prompt() function to get input from the user. The prompt should include the entire chat history.
  2. Use the JavaScript alert() function to display any errors that occur when calling the OpenAI API.
  3. Use the JavaScript fetch() function to send POST requests to the OpenAI API's chat endpoint.
  4. Create and send a JSON object in the body of the request that includes an array of message objects. Each message object should have a 'role' and a 'content' property.
  5. Handle the response from the OpenAI API. If there is a message in the response, add it to the chat history and continue the chat. If there is no message, alert the user that the assistant did not return a response.
  6. The chat should continue until the user enters an empty input or clicks "Cancel".

Expected Output:

The application should provide an interactive chat experience in the browser, with the user inputting messages via the prompt() function and the AI's responses being added to the chat history. If the AI does not return a message, the user should be alerted that no response was received.

Hint:

Remember to use the 'Authorization' header with the value `Bearer ${OPEN_AI_API_KEY}` for your API requests. Replace `${OPEN_AI_API_KEY}` with your actual OpenAI API key.
